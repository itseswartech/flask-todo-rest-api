# Flask To-Do REST API

## Overview

A RESTful Task Management API developed using Python, Flask, and SQLite.

## Features

- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks

## Technologies

- Python
- Flask
- SQLite
- REST API
- JSON

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /tasks | Create Task |
| GET | /tasks | View Tasks |
| PUT | /tasks/<id> | Update Task |
| DELETE | /tasks/<id> | Delete Task |

## Installation

pip install -r requirements.txt

python app.py

http://127.0.0.1:5000
